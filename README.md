## Service Registry do SpringBoot

### Executar via terminal
`$ mvn spring-boot:run`

`$ mvn spring-boot:run -Dspring-boot.run.arguments=--server.port=8085`


### Acesso ao Dashboard Eureka
http://127.0.0.1:8761/ ou http://localhost:8761/login
